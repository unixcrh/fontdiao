FontAwesome 的中文分支

可与 FontAwesome 同时使用，对中文网站更加友好。

完美支持 Retina 屏幕

没有 @2x 的图片，不需要 JavaScript，因为使用了矢量字体。另外次像素平滑技术将使你的图标更加清晰锐利。

广泛的应用场合

想要在某个淘宝链接前嵌入淘宝的图标？但是不同颜色或大小的文字需要好多图片，Font屌 是更好的选择。

一个字体包含 20 多个图标

客户端浏览器只下载一个字体，就能拥有 20 多个常用网站的图标。

可用于商业应用

Font屌 的字体和样式表可用于商业应用，且完全免费。

支持 IE7

和 FontAwesome 一样，Font屌 v0.0.1 支持 IE7。

为 Twitter Bootstrap 而生

为了解放网站前端屌丝码农，我们必须支持 Twitter Bootstrap 2.0.

CSS 控制

你可以非常方便地控制图标的颜色、大小、阴影等所有可以用 CSS 控制的属性。

无限缩放

使用矢量图形的好处是无论怎样缩放它都能得到完美的视觉效果。

集成

Font屌 与 Twitter Bootstrap 同时使用是一件很简单的事, 单独使用 Font屌 也很方便。

与 FontAwesome 一起使用

当你使用 Twitter Bootstrap 和 FontAwesome 时使用此方法。

  1. 先在 Bootstrap 中集成 FontAwesome。
  2. 将 Font屌 的 font 目录复制到你的项目中。
  3. 复制 fontdiao.css 到你的项目中。
  4. 打开项目中的 fontdiao.css 确定字体文件的 url 指向正确的位置。
  5. 在你的 html <head> 标签中，引用 fontdiao.css。 
    <link rel="stylesheet" href="../css/bootstrap.css">
<link rel="stylesheet" href="../css/font-awesome.css">
<link rel="stylesheet" href="../css/fontdiao.css">

不和 Bootstrap 一起用？

Font屌 在没有 Twitter Bootstrap 的情况下同样可以用。

  1. 将 Font屌 的 font 目录复制到你的项目中。
  2. 复制 fontdiao.css 到你的项目中。
  3. 打开项目中的 fontdiao.css 确定字体文件的 url 指向正确的位置。

代码

同一行中的图标

使用 <i> 标签在任何位置插入 Font屌 图标。

icon-buluo 
<div style="font-size: 24px;">
  <i class="icon-buluo"></i> icon-buluo
</div>
图标的样式将通过 CSS 的 :before 选择器输出。

通过使用 icon-large 类改变图标大小。这将使图标在其容器内变大 33%。

icon-buluo 
<div style="font-size: 24px;">
  <i class="icon-buluo icon-large"></i> icon-buluo
</div>
